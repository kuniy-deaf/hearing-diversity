# 聞こえ方の違いを体験する MVP

GitHub Pages で公開できる静的Webアプリの試作です。

## 機能

- 音声ファイルをブラウザ内で読み込み
- オージオグラムタイプを選択
- サンプル図と説明文を表示
- 英語ベースの簡易スピーチバナナ表示
- 音量スライダー
- ことばの明瞭度スライダー
- 元音声と加工音声の比較再生
- `noindex,nofollow` と `robots.txt` による検索エンジン除外指定

## 注意

このデモは医学的な聴覚シミュレーションではありません。

目的は、以下を伝えることです。

- 聴力レベルが同じでも聞こえ方は人によって違う
- 音が聞こえることと、ことばとして理解できることは同じではない
- あなたが体験したのは、数ある聞こえ方のひとつである
- 同じ難聴者でも、聞こえ方は一人ひとり異なる

## GitHub Pages で公開する手順

1. GitHubで新しいリポジトリを作成
2. このフォルダ内のファイルをアップロード
3. Settings → Pages
4. Source: Deploy from a branch
5. Branch: main / root を選択
6. 数分待つ
7. `https://ユーザー名.github.io/リポジトリ名/` にアクセス

## スピーチバナナについて

このMVPでは英語音声を基準としたスピーチバナナを簡略化して表示しています。臨床的な正確性より、一般向けに「周波数帯と聞こえやすい音の関係」を直感的に伝えることを目的としています。
