# Hearing Diversity

A sample web application for exploring different ways of hearing.

The hearing experience in this app is only one example.

Even among people with hearing loss, hearing experiences vary from person to person.

This project explores how factors such as audiogram shape, loudness, and speech clarity can affect hearing.

## About

This application is intended as a starting point for understanding hearing diversity.

It is not designed to accurately reproduce any specific person's hearing experience.

Actual hearing experiences may differ depending on many factors, including:

* Hearing thresholds
* Audiogram configuration
* Speech clarity
* Hearing aids or cochlear implants
* Listening environment
* Left-right differences

## Features

* Audio file upload
* Audiogram-based hearing profiles
* Speech banana visualization
* Loudness adjustment
* Speech clarity adjustment
